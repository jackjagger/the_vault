# the_vault

This is where I store all my information on the topics that I study whether that be for university, self-study or any other piece of interesting knowledge.

## How I use the_vault

- Read information from a source material
- Cite sources
- Write own interpretation of source material
- Connect to other pieces of information based on what I've read (Encoding)
- Retrieve information by pulling up a random note and seeing if I can make any improvements.
- Push to GitHub
